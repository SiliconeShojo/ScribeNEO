# 🖋️ ScribeNEO: The Ultimate Prompting Hub

ScribeNEO is a professional-grade prompt engineering extension for Stable Diffusion Forge, designed to bridge the gap between simple ideas and high-fidelity image generation. With an integrated **Vision Toolset** and a dynamic **Persona Lab**, ScribeNEO transforms your workflow into a semantic dialogue with your favorite AI models.

![ScribeNEO Dashboard](Screenshot.png)

> [!IMPORTANT]
> **Compatibility Notice:**
> This extension has been developed and tested exclusively on [Forge Neo](https://github.com/Haoming02/sd-webui-forge-classic/tree/neo). While it may function on other Forge forks or A1111, there is no intention to test or provide support for environments other than Forge Neo.

---

## ✨ Key Features

### 🌪️ Scribe Prompt Enhancer
Transform keywords into rich, natural language descriptors. Support for:
- **OpenRouter & Hugging Face**: Cloud-based power for complex prose.
- **Local Ollama**: Privacy-first enhancement using your own hardware (Llama 3, Mistral, etc.).
- **Smart Personas**: Instant switching between Prose Writers, Tag Specialists, and more.

### 👁️ Vision Toolset
Scan and "undress" image metadata into actionable prompt components:
- **Image-to-Prose**: Deep visual analysis using high-end vision models.
- **Image-to-Tags**: Precise Booru/Danbooru tagging for stable diffusion training and prompting.
- **Interchangeable Engines**: Use Gemini 2.0, LLaVA, or GPT-4o with ease.

### 🎭 Persona Lab
Craft and curate the "Soul" of your AI. Define custom system prompts that dictate how your AI interprets your intents.
- **Zero-Friction Creation**: Save and manage multiple prompt personalities.
- **Persistent Memory**: ScribeNEO remembers your last selected persona across sessions.

## 🚀 Installation

1.  Open your **Stable Diffusion WebUI** (Forge Neo).
2.  Navigate to the **Extensions** tab.
3.  Go to **Install from URL**.
4.  Paste this repository link: `https://github.com/SiliconeShojo/ScribeNEO.git`
5.  Click **Install** and restart the WebUI.

---

## 🗝️ Getting API Keys

To use ScribeNEO with cloud providers, you will need an API key:

- **OpenRouter (Recommended)**: Create an account and generate a key at [openrouter.ai](https://openrouter.ai/keys). OpenRouter gives you access to the widest variety of models (Gemini, Claude, GPT-4o) in one place.
- **Hugging Face**: Generate an "Access Token" (with internal/read permissions) at [huggingface.co/settings/tokens](https://huggingface.co/settings/tokens).
- **Ollama**: No API key required! Ollama runs locally on your machine. Just ensure the Ollama server is running.

---

## 🛠️ Configuration

1.  Open the **ScribeNEO** tab.
2.  Expand the **🛠️ SCRIBE HUB** section.
3.  Select your provider (OpenRouter is recommended for the best results).
4.  Enter your **API Key** and click **💾 Save Configuration**.
5.  Use the **🔄 Sync** buttons in the Enhancer or Vision modules to fetch your models.
